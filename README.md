## рҹ§  Overview

**tech2** is a full-stack web application combining a blazing-fast React + TypeScript frontend with a Python backend. It integrates Google's Gemini AI SDK for intelligent features, and is built on a modern component architecture using shadcn/ui, TanStack Router, and Tailwind CSS v4.

The project is developed on the [Lovable](https://lovable.dev) platform вҖ” a collaborative AI-powered development environment that keeps your code and editor in sync.

## рҹҡҖ Getting Started

### вң… Prerequisites

- **Node.js 18+** or **[Bun](https://bun.sh/)** (recommended)
- **Python 3.12+**
- **Git**

### рҹ“Ҙ 1. Clone the Repository

```bash
git clone https://github.com/takvir-tur/tech2.git
cd tech2
```

### рҹ“Ұ 2. Install Frontend Dependencies

```bash
# Using Bun (recommended)
bun install

# Or using npm
npm install
```

### рҹ§Ә 3. Start the Development Server

```bash
# Using Bun
bun run dev

# Using npm
npm run dev
```

The app will be available at **http://localhost:5173**

### рҹҸ—пёҸ 4. Build for Production

```bash
bun run build
# or
npm run build
```

### рҹ‘Җ 5. Preview Production Build

```bash
bun run preview
```
